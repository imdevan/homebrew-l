# Homebrew Tap for l

This is the official Homebrew tap for [l](https://devan.gg/l).

The formula is named `l` but installs the binary as `l`.

## Installation

```bash
brew tap imdevan/l
brew install l
```

## Usage

After installation, use the `l` command:

```bash
l --help
```

## Updating

```bash
brew update
brew upgrade l
```

## Uninstall

```bash
brew uninstall l
brew untap imdevan/l
```
